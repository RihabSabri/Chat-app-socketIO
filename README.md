<h1>Real Time Chat App</h1>

<h2>This Application was built using </h2>
<ul>
<li>React Js</li>
<li>Express Js</li>
<li>Socket IO</li>
</ul>

<h2> To run app  </h2>
<ol>
<li>Clone repository</li>
<li>install dependencies with <code>npm i </code> or <code>yarn</code> if using yarn</li>
<li>run <code>npm start</code> </li>
</ul>
